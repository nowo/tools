repeat重复、遍历

指定数组项的重复次数。可重复数组必须只包含两个项目：第一个是重复标签，第二个是必须被重复的项目。如果没有指定参数，项目将被重复0到10次。如果指定了最小参数，项目将被重复该次数。如果指定了两个参数，项目将在指定的范围内被重复。

使用

{{repeat([min], [max])}}

属性

参数 类型 介绍
min (可选) Number 范围内的最小数字。默认为0。
max (可选) Number 范围内的最大数字。默认为10。

返回

Number

使用方法

JSON 生成器有一个方便的语法。你可以为你在左边框中看到的模板指定JavaScript对象。它将被无限深入地克隆。字段可以有任何名称和值，但它必须是有效的JavaScript。值，是字符串，可以包含一个或多个模板标签。当你点击 "生成 "时，数据源对象将被复制几次，标签的位置将被插入随机值。

你可以通过点击 "复制到剪贴板 "将生成的JSON复制到剪贴板上。你也可以通过点击 "下载 "按钮下载生成的文件。生成的JSON大小出现在有生成数据的领域的右上方。

你可以从下拉列表中为生成的JSON选择缩进。它的值可以是 "紧凑"、2 tab 符、3 tab 符和4 tab 符。

JSON Generator 会记住您在 localStorage 中的最后一个模板。如果您想重置它，请点击 "重置 "按钮来重置模板和用户界面。
